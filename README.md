# Esp32-Temp-Sensoring
Sistema de aquisição de dados e monitoramento térmico via Web Server com ESP32. Integra sensores heterogêneos (Termopar Tipo K com MAX6675 e Termistor NTC 100k) aplicando a equação de Steinhart-Hart. Desenvolvido para a disciplina de Sensores e Instrumentação do IFSP - Câmpus Bragança Paulista.
